# [Dongguk Univ AI Graduate Calculator for U] -> DAGCU

- local setup
  ```
  npm install react 
  npm install react-router-dom 
  ```
  
- backend/.env.sample의 내용 수정
  ```
  DB_HOST=localhost
  DB_USER="yourname"
  DB_PASSWORD="yourpassword"
  DB_NAME="yourdbname"
  JWT_SECRET=1
  PORT=5001 
  ```
- 이후 파일명 env로 변경
